아래 위치에서 최신 설치본만 받아 설치합니다.

- GitHub Releases: [https://github.com/roycekim84/worklog/releases/latest](https://github.com/roycekim84/worklog/releases/latest)

권장 파일:
- macOS (Apple Silicon): `Worklog-<version>-arm64.dmg` 또는 `Worklog-<version>-arm64-mac.zip`
