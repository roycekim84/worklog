# Latest Installer

회사 PC는 Windows 기준으로 사용합니다.

권장 경로:
- 가능하면 설치형보다 `npm install` 후 `npm run web`로 실행하세요.
- 브라우저 접속 주소: `http://127.0.0.1:3210`

아래 위치에서 최신 설치본만 받아 설치합니다.

- GitHub Releases: [https://github.com/roycekim84/worklog/releases/latest](https://github.com/roycekim84/worklog/releases/latest)

권장 파일:
- Windows installer: `Worklog Setup <version>.exe`
- Windows portable zip: `Worklog-<version>-arm64-win.zip`

현재 빌드 확인 파일:
- `release/Worklog Setup 0.1.0.exe`
- `release/Worklog-0.1.0-arm64-win.zip`
